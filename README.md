# 🏎️ Clássico Automobilismo

[![Navegar no Projeto](https://img.shields.io/badge/Acessar-Deploy-blue?style=for-the-badge&logo=vercel)](https://cl-ssico-automobilismo.vercel.app/login)

Bem-vindo ao **Clássico Automobilismo**, uma plataforma digital desenvolvida para entusiastas de veículos de alta performance. O projeto funciona como um *showroom* interativo, permitindo que os usuários explorem modelos detalhados, realizem buscas refinadas e gerenciem seus perfis personalizados.

---

## 🚀 Descrição do Projeto

O projeto consiste em uma plataforma de *showroom* digital voltada para o mercado de automóveis clássicos. A aplicação foi projetada para oferecer uma experiência de navegação fluida e intuitiva.

**Destaques técnicos:**
- **Front-end:** Desenvolvido em **React.js** com **Vite** e estilizado com **Tailwind CSS**.
- **Gestão de Dados:** Integração com **JSON Server** (hospedado no Render) para autenticação e persistência de usuários.
- **Performance:** Imagens servidas via **CDN (jsDelivr)**, garantindo carregamento rápido e otimizado.
- **Segurança:** Autenticação protegida, validação de campos em tempo real e rotas protegidas.

---

## ⚠️ Nota sobre Performance (Infraestrutura)
A aplicação utiliza o plano gratuito do **Render.com** para o back-end. Devido à natureza desse serviço, o servidor entra em modo de suspensão após períodos de inatividade. 
* **Primeiro acesso:** O carregamento inicial pode levar de **30 a 45 segundos** enquanto o servidor "desperta". Após esse tempo, a navegação será rápida e fluida.

## 🛠️ Tecnologias
- **Framework:** [React](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Estilização:** [Tailwind CSS](https://tailwindcss.com/)
- **Comunicação:** [Axios](https://axios-http.com/)
- **Servidor:** [JSON Server](https://github.com/typicode/json-server)
- **Hospedagem:** [Vercel](https://vercel.com/) (Front) e [Render](https://render.com/) (Back)

   cd Cl-ssico-Automobilismo
2. **Instale as dependências** 
   ```bash

3. **Inicie o servidor de desenvolvimento**
   ```bash

4. **Acesso:**
Após rodar o comando, o terminal exibirá um link (geralmente http://localhost:5173). Abra-o no seu navegador.
